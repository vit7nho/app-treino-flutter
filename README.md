
### Scripts

|                    Script | Target                                             |
| ------------------------- | -------------------------------------------------- |
|                    `npm run dev` | Run API in **development** environment      |
|                    `npm start` | Run API in **production** environment         |
|                    `npm run migrate` | Create database tables                  |
|                    `npm run seed` | Populate database tables                   |
